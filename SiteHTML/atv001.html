⌞Head⌝࣪

O head no HTML é a seção que contém informações sobre a página, mas que não são exibidas diretamente no conteúdo. Ele inclui metadados, como o título da página, links para folhas de estilo link, scripts script, definições de codificação e outras configurações importantes para o funcionamento e aparência do site

⌞ Body⌝

O body no HTML é a parte do código onde fica o conteúdo visível da página, como textos, imagens, vídeos, botões e links. Tudo o que o usuário vê e interage no site está dentro dessa seção. Ele funciona como o "corpo" da página.

⌞Lang ⌝

O atributo lang no HTML define o idioma principal do conteúdo da página. Ele é usado na tag html para ajudar navegadores, leitores de tela e mecanismos de busca a entenderem o idioma da página.

⌞Tag P ⌝

O p em HTML é a tag usada para criar parágrafos de texto. Ele organiza o conteúdo em blocos, dando espaçamento automático antes e depois de cada parágrafo. É essencial para a estruturação de textos em uma página web.

⌞A tag B⌝

A tag b em HTML é usada para aplicar negrito ao texto, sem adicionar ênfase semântica. Ela apenas altera a aparência visual, tornando o texto mais destacado.

⌞br⌝

O br em HTML é a tag usada para quebrar linhas dentro de um texto. Diferente do p, ele não cria um novo parágrafo, apenas força a quebra para a linha seguinte. É útil para endereços, poemas ou qualquer situação onde a quebra de linha seja necessária sem espaçamento extra.

⌞Sub e Surp⌝࣪

As tags sub e sup em HTML são utilizadas para formatar texto em subscrito e sobrescrito, respectivamente. A tag sub reduz o tamanho do texto e o posiciona abaixo da linha base, sendo comumente usada para fórmulas químicas. Já a tag sup eleva o texto acima da linha base, sendo útil para expoentes matemáticos.

⌞Style ⌝

O style em HTML é um atributo ou uma tag usada para adicionar estilos ao conteúdo da página. Ele pode ser aplicado diretamente em elementos com o atributo inline ou dentro da tag style no head para definir regras CSS que afetam vários elementos da página.

⌞Img ⌝

A tag img em HTML é usada para incorporar imagens em uma página web. Ela não possui uma tag de fechamento e é definida com o atributo src para especificar o caminho da imagem. Além disso, o atributo alt deve ser usado para descrever a imagem, melhorando a acessibilidade e o SEO.

⌞Links ⌝

links em HTML são criados com a tag e servem para conectar páginas, documentos ou seções dentro de uma página. O principal atributo é href, que define o destino do link. Outros atributos úteis incluem target, que controla onde o link será aberto (_self na mesma aba e _blank em uma nova aba), rel, que define a relação entre as páginas (noopener noreferrer melhora a segurança e nofollow impede que buscadores sigam o link), title, que exibe um texto ao passar o mouse, download, que permite baixar arquivos, e id/name, usados para criar âncoras internas. Links podem ser estilizados com CSS para alterar sua aparência.

⌞list⌝

As listas em HTML organizam itens de forma estruturada e podem ser ordenadas (ol) ou não ordenadas (ul). Cada item da lista é definido pela tag li. As listas ordenadas usam números ou letras automaticamente, enquanto as não ordenadas exibem marcadores. O atributo type pode personalizar os marcadores (circle, square, disc) e a numeração (1, A, a, I, i). O atributo start define o número inicial em listas ordenadas, e reversed inverte a ordem.
